# Welcome to My Portfolio

I'm excited to share my new portfolio website with you. This site is inspired by the beautiful HD-2D aesthetic of games like Octopath Traveler, featuring soft lighting, layered depth, and calm motion.

## Design Philosophy

The design emphasizes:

- **Minimalism**: Clean layouts with purposeful whitespace
- **Typography**: Readable, well-structured text content
- **Motion**: Subtle, ambient animations that enhance rather than distract
- **Performance**: Static-first architecture with minimal JavaScript

## Technical Stack

This site is built with:

- **Astro**: For static site generation
- **TypeScript**: For type-safe development
- **CSS Custom Properties**: For a maintainable design system
- **Markdown**: For content authoring
